# 依樣畫葫蘆

—— 尤貴弘，數學老師

在數學課聽到這句話絕對不是叫你用相同的方法解另一道題目，而是老師準備要大展繪畫長才了？！老師在班上常常教我們畫葫蘆，每個步驟、細節都非常講究。甚至揚言段考要考畫葫蘆呢！

# the偷

老師上課喜歡丟題目讓我們互相討論練習，最後再抽人上台解題。這種活動通常不知不覺就耗掉了一兩節課，難怪會被我們譽為專業的ㄒㄕ小偷啊！

# 尤達大師

多年前被學生 P 成尤達大師的圖，至今仍廣為流傳，你們找得出來嗎？
p.s. Google是好用的搜尋工具


# （手比心

—— 何宣螢，物理老師

待在物理組的同學們最能體會了，老師的訊息兩三句就會出現一次這樣生動的註解，可說是老師的招牌啊！

# 少子化改善專家

我們剛進高一時，老師請了一學期的產假，如今高二下又請了一次段考的產假生第二胎。像老是這樣致力改善少子化問題的精神實在令我們敬畏。
所以我說 @張杰 你問出是男是女了沒

# 段考獨立命題

想當初高一老師請假，段考和高二一起考的歡樂時期，相比現在每次都獨立命題的段考，唉，美好的時光總是過的那麼快，現在班平均能及格就是福了…


# 哎呀

—— 柯淑惠，化學老師

通常聽到這個可能是你上課不認真被抓到或者上台解題出糗，老師幫你圓場時的開頭，不過最慘的可能是老師要開始嘴人的預言…

# 真~的~

這真誠的一句話講出來似乎意有所指呢，不知道是誰教壞老師的？


# 豬哥啊

—— 劉玉山，生物老師 / 班級導師

無論是你上課睡覺、答錯問題、或者又幹了什麼蠢事，總是聽的到班導一句「豬哥啊！」。雖然說是罵人的話，但聽到時全班總是笑得合不攏嘴呢！

# 打叉叉，不要理他

身為科學班的生物老師，上課當然會補充大量的課外知識。為了釐清，總會冒出一句「打叉叉，不要理他」，也因此成為繼「豬哥啊」後，大家競相模仿的一句話。@林尚廷 就是說你

# 電子白板

自從換上電子白板後，老師上起課來就變得非常艱辛，對觸控白板總是無法操控自如，常常抱怨學校經費花錯地方。希望老師能趕快熟悉他呀！


# 記得關冷氣

—— 林淑玲，資訊老師

資訊組的同學們常常借用教室到很晚，老師總得提醒健忘的我們記得要關冷氣…但依然，他們早已浪費不知道幾個晚上的冷氣費了…
